Progetto:

    Chatroom

Sviluppatore:

    Zhu Donglai

Descrizione:

    Chatroom è un progetto dove gli utenti si possono comunicare fra di loro in una aula virtuale.
    Server TCP - gestione delle connessioni dei client e permette il ricevimento e invio dei messaggi fra i client
    Client testuale - comunicazione fra client in terminale
    Client mobile - comunicazione fra client in una interfaccia grafica basata sul client testuale

Scelte di sviluppo:

    Flutter - linguaggio richiesto
    Gestione nomi client - il server associa a ogni socket il nome dell'utente, sapendo così chi si connette e disconnette.
    Formato messaggio - 'user: msg'
